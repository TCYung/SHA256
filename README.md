## Project to learn about Bitcoin's PoW and how hash rate can be increased 

ControlTestSV - Converts 640 bit input into 256 bit SHA256 hash

BitcoinSHA256 - Executes the Double SHA256 algorithm to produce a Bitcoin hash

PipelinedBTCSHA256 - Pipelined version of BitcoinSHA256, calculates 3 Bitcoin hashes simultaneously 

TargetChecker - Converts Bitcoin numerical difficulty value to a hash target value that a valid hash must be less than. Used to check if a valid hash has been found
